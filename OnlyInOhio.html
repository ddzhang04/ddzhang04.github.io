Only In Ohio Backrooms

"Only in Ohio Backrooms!" is a Unity-based survival-horror game infused with internet meme culture, built using C# and leveraging NavMesh AI pathfinding, object interaction, and dynamic UI systems. The game challenges players to collect 10 fries scattered throughout a surreal McDonald's-Backrooms hybrid environment while avoiding an Caseoh. The enemy AI utilizes NavMeshAgents to track the player in real time, creating an engaging chase mechanic. This project was made with Anthony Zhao for the Brainrot Hackathon.
